Module rpgplus.scheduler

Provides scheduler functions.

Functions

async (fn) Creates a new function that will call the given function asynchronously when invoked.
sync (fn) Creates a new function that will run the given function in the Bukkit main thread when invoked.
delay (delay, fn) Calls a function after a given delay.
delayed (delay, fn) Creates a new function that will invoke the given function after the given delay when called.
repeating ([delay=0], period, fn) Starts to periodically call the given function after the given delay with the given period.
cancel (taskId) Cancels the task with the given ID.


Functions

async (fn)
Creates a new function that will call the given function asynchronously when invoked. Parameters are passed through.

Parameters:

  • fn function that should be called asynchronously

Returns:

    a new function
sync (fn)
Creates a new function that will run the given function in the Bukkit main thread when invoked. Parameters are passed througgh. This function only needs to be used inside asynchronous code.

Parameters:

  • fn function that should run in the main thread
delay (delay, fn)
Calls a function after a given delay.

Parameters:

  • delay delay in ticks
  • fn function to call
delayed (delay, fn)
Creates a new function that will invoke the given function after the given delay when called. Parameters are passed through.

Parameters:

  • delay delay in ticks
  • fn function to call
repeating ([delay=0], period, fn)
Starts to periodically call the given function after the given delay with the given period.

Parameters:

  • delay delay before first execution in ticks (default 0)
  • period period in ticks
  • fn function to call periodically

Returns:

    a task ID that can be used by cancel to cancel the calls
