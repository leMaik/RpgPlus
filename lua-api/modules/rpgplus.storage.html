Module rpgplus.storage

Provides a simple key-value storage.

Functions

get (key[, defaultValue=nil]) Get the value of a specific key or the default value, if the key doesn't exist.
set (key, value) Set the value of the given key.
of (player) Get the storage of the given player.


Functions

get (key[, defaultValue=nil])
Get the value of a specific key or the default value, if the key doesn't exist.

Parameters:

  • key key
  • defaultValue fallback value (default nil)

Returns:

    value of the given key or the fallback value if the key doesn't exist
set (key, value)
Set the value of the given key. If the value is a table, subkeys will be set based on the keys of the table, recursively. Set value to nil to remove the key. This will also remove subkeys.

Parameters:

  • key key
  • value value
of (player)
Get the storage of the given player.

Parameters:

  • player a player name

Returns:

    storage of the given player, it works like this storage but is player-specific
