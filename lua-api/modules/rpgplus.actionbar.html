Module rpgplus.actionbar

Provides functions to set the action bar of players.

Functions

set (player, message[, duration]) Sets the given message as action bar for the given player.
clear (player) Clears the action bar for the given player.


Functions

set (player, message[, duration])
Sets the given message as action bar for the given player.

Parameters:

  • player player
  • message message of the action bar, formattings are possible using & as escape character
  • duration duration, in ticks, that the action bar should be displayed - if not specified, the action bar will be displayed forever (optional)
clear (player)
Clears the action bar for the given player.

Parameters:

  • player player
