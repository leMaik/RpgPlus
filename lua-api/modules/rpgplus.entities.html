Module rpgplus.entities

Provides functions to spawn entities and work with them.

Functions

Functions

spawn (type, options)
Registers and creates a managed entity.

Parameters:

  • type the type of the entity to spawn
  • options a table with options for the entity

Returns:

    Entity the spawned entity
getNearby (location, radius)
Gets the entities in the given radius near the given location. This will only get entities that were spawned with spawn.

Parameters:

  • location center location
  • radius radius around that location, in blocks

Returns:

    Entity[] list of nearby entities
