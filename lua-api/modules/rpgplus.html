Parameters:

  • level level of the message, must be info, warn or error (default "info")

spawn (type, options)
Deprecated, use rpgplus.entities.spawn instead.

Registers and creates a managed entity.

Parameters:

  • type the type of the entity to spawn
  • options a table with options for the entity

Returns:

    Entity the spawned entity
command (command, handler)
Registers a command.

Parameters:

  • command the command, either a string or a list of the command and all nested commands
  • handler a function that handles the command, it is invoked with the sender, the command and the arguments. Note that if the command is a nested command, the command parameter is the last command and the arguments are anything the sender entered after that command
playercommand (command, handler)
Registers a command that can only be executed by a player (not by the console or by a command block).

Parameters:

  • command the command, either a string or a list of the command and all nested commands
  • handler a function that handles the command, it is invoked with the sender, the command and the arguments. Note that if the command is a nested command, the command parameter is the last command and the arguments are anything the sender entered after that command
sendMessage (players, ...)
Sends a message to the given players.

Parameters:

  • players a list of players to send the message to, if it is only one player, no list is required
  • ... messages to send to the given players
broadcastMessage (...)
Broadcasts a message to all players.

Parameters:

  • ... messages to send to all players

once (event, handler)
Register an event handler that is only called once.

Parameters:

  • event event to add a one-time handler to
  • handler a handler function that is invoked whenever the event happens, gets the event as first parameter
off (event[, handler])
Remove an event handler.

Parameters:

  • event event to remove an event handler from
  • handler the handler function to remove as previously added with on; if not specified, all handlers of this event will be removed (optional)

Returns:

    true if any event handler was removed, false if not
