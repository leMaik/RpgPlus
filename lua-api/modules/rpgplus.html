Functions

log ([level="info"], message)
Writes a message into the log.

Parameters:

  • level level of the message, must be info, warn or error (default "info")
  • message message to log
registerVillager (options, interact)
Registers and creates a managed villager.

Parameters:

  • options a table with options for the villager
  • interact a function that is invoked when a player right-clicks the villager
navigateTo (entity, destination[, options[, callback]])
Navigates an entity to the given location, using A*.

Parameters:

  • entity entity to navigate
  • destination destination, a table with x, y and z fields
  • options a table with options for pathing, options are openDoors (boolean), openFenceGates (boolean) and speed (int, in meters per second) (optional)
  • callback callback that is invoked when the entity has reached the destination (optional)

Returns:

    true if the navigation was possible, false if not path was found
command (command, handler)
Registers a command.

Parameters:

  • command the command, either a string or a list of the command and all nested commands
  • handler a function that handles the command, it is invoked with the sender, the command and the arguments. Note that if the command is a nested command, the command parameter is the last command and the arguments are anything the sender entered after that command
playercommand (command, handler)
Registers a command that can only be executed by a player (not by the console or by a command block).

Parameters:

  • command the command, either a string or a list of the command and all nested commands
  • handler a function that handles the command, it is invoked with the sender, the command and the arguments. Note that if the command is a nested command, the command parameter is the last command and the arguments are anything the sender entered after that command
sendMessage (players, ...)
Sends a message to the given players.

Parameters:

  • players a list of players to send the message to, if it is only one player, no list is required
  • ... messages to send to the given players
broadcastMessage (...)
Broadcasts a message to all players.

Parameters:
