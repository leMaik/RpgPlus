Module rpgplus.types

This pseudo-module defines tables that are used as parameters in various functions.

Neither this module nor its fields actually exist. The fields of this module are only used to document tables that various functions require.

Fields

itemstack A normal table that defines an item stack.


Fields

itemstack
A normal table that defines an item stack. I.e. it is used in the Inventory API.
  • type type of the item, must be a Bukkit Material
  • data data of the item, if nil, the data is ignored when matching and set to 0 when giving the item to a player (default nil)
  • amount amount of the item (default 1)
  • name item name (optional)
  • lore list of lines for the lore of this item (optional)
  • unbreakable whether the item should be unbreakable (default false)
  • texture base64-encoded texture for skull items (only works with type=skull_item, data=3) (optional)
