create (path[, width=1[, height=1]])
Creates a new poster. This rescales the original image to fit the specified dimensions.

Parameters:

  • path path of the poster, relativ to the script directory
  • width width of the poster, in blocks (default 1)
  • height height of the poster, in blocks (default 1)

Returns:

    Poster the new poster
