Functions

hasItems (player, ...)
Checks if the player has the given items.

Parameters:

  • player player
  • ... items to check for (each as an itemstack)

Returns:

    true if the player has the items or false if not
giveItems (player, ...)
Gives items to a player.

Parameters:

  • player player
  • ... items to give to the player (each as an itemstack)
takeItems (player, ...)
Takes items from a player. The items are taken even if they are not enough.

Parameters:

  • player player
  • ... items to take from the player (each as an itemstack)

Returns:

    missing amount of each specified item, as multiple return values

Tables

itemstack
A table that represents an itemstack.

Fields:

  • type type of the item, must be a Bukkit Material
  • data of the item, if nil, the data is ignored when matching and set to 0 when giving the item to a player (default nil)
  • amount amount of the item (default 1)
