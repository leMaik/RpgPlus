Module rpgplus.inventory

Provides inventory functions.

Functions

hasItems (player, ...)
Checks if the player has the given items.

Parameters:

  • player player
  • ... items to check for (each as an rpgplus.types.itemstack or a string of the form id:data:amount)

Returns:

    true if the player has the items or false if not
giveItems (player, ...)
Gives items to a player.

Parameters:

  • player player
  • ... items to give to the player (each as an rpgplus.types.itemstack or a string of the form id:data:amount)

Returns:

    amount of each specified item that didn't fit into the inventory, as multiple return values
takeItems (player, ...)
Takes items from a player. The items are taken even if they are not enough.

Parameters:

  • player player
  • ... items to take from the player (each as an rpgplus.types.itemstack or a string of the form id:data:amount)

Returns:

    missing amount of each specified item, as multiple return values
openChest (player, size, title, items)
Opens an inventory.

Parameters:

  • player player
  • size size of the inventory (rows of 9)
  • title title of the inventory
  • items table of items in the inventory, with slots as keys

Returns:

    Inventory inventory
