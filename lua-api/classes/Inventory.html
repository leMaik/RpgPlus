Class Inventory

An inventory.

Methods

Inventory:setItems (items[, clear=true]) Sets the items in this inventory.
Inventory:setItem (slot, item) Sets the item in the given slot of this inventory.
Inventory:open (player) Opens this inventory for a player.
Inventory:close () Closes this inventory for all viewers.
Inventory:on (event, handler) Add a callback for the given event.
Inventory:once (event, handler) Register an event handler that is only called once.
Inventory:off ([event[, handler]]) Remove an event handler.


Methods

Inventory:setItems (items[, clear=true])
Sets the items in this inventory.

Parameters:

  • items table of items in the inventory, with slots as keys
  • clear whether the inventory should be cleared before adding the items (default true)
Inventory:setItem (slot, item)
Sets the item in the given slot of this inventory.

Parameters:

  • slot slot
  • item item to put into the slot
Inventory:open (player)
Opens this inventory for a player.

Parameters:

  • player player
Inventory:close ()
Closes this inventory for all viewers.
Inventory:on (event, handler)
Add a callback for the given event.

Parameters:

  • event

    event to add a handler to, possible events are:

    • "click" - InventoryClickEvent
    • "close" - InventoryCloseEvent
  • handler a handler function that is invoked whenever the event happens, gets the event as first parameter
Inventory:once (event, handler)
Register an event handler that is only called once.

Parameters:

  • event event to add a one-time handler to
  • handler a handler function that is invoked whenever the event happens, gets the event as first parameter
Inventory:off ([event[, handler]])
Remove an event handler.

Parameters:

  • event event to remove an event handler from; if not specified, all event handlers of this inventory will be removed (optional)
  • handler the handler function to remove as previously added with on; if not specified, all handlers of this event will be removed (optional)

Returns:

    true if any event handler was removed, false if not
