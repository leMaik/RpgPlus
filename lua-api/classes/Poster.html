Fields

Poster.width
Width of this poster, in blocks. This field is read-only.
Poster.height
Height of this poster, in blocks. This field is read-only.

Methods

Poster:giveTo (player[, callback])
Gives all items of this poster to the given player, as maps.

Parameters:

  • player player
  • callback callback that is invoked after giving the items to the player, parameter is a boolean that is false if giving the items failed and true otherwise (optional)
Poster:attach (location[, callback])
Attaches this poster to a wall. Note that a poster may be attached multiple times.

Parameters:

  • location location of the top left block to attach this poster to, including a face field that is either north, east, south or west
  • callback callback that is invoked after attaching the poster, parameter is a boolean that is false if attaching the poster failed and true otherwise (optional)
